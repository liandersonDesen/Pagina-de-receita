# 🍛 Receita de Feijoada Completa

Este projeto é uma página web elegante e interativa que apresenta a receita da tradicional feijoada brasileira. Desenvolvido com foco em **HTML Semântico** e **CSS Avançado**, o projeto destaca o uso da tipografia clássica e recursos de interatividade para o usuário.

## ✨ Funcionalidades

* **Checklist Interativo**: O usuário pode marcar os ingredientes conforme os prepara.
* **Feedback Visual**: Ao marcar um item, o texto é automaticamente riscado e a cor é alterada para indicar conclusão.
* **Design Gastronômico**: Paleta de cores baseada em tons de marrom e laranja para uma experiência imersiva.
* **Tipografia Clássica**: Uso da fonte "Alice" via Google Fonts para um visual de livro de receitas.

## 🛠️ Tecnologias

* **HTML5**: Estrutura semântica.
* **CSS3**: Estilização, efeitos de hover e estados `:checked`.
* **Google Fonts**: Integração da fonte 'Alice'.

## 📖 Sobre a Receita

A feijoada brasileira é um cozido de feijão preto com carnes salgadas e defumadas. O processo envolve:
* **Dessalga (24h)**: Preparo prévio das carnes.
* **Cozimento Lento**: Para garantir a maciez das carnes e o caldo grosso.
* **Refogado Especial**: Com bacon, alho e cebola.
